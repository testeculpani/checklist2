Observações

  • Esta versão usa uma leitura mais estável do PDF com ordenação por linhas.
  • Se a nota for PDF escaneado como imagem pura, pode não extrair bem.
  • Nesse caso será preciso OCR depois.
  • Para DANFE comum de fornecedor, normalmente funciona melhor.
  • O botão Mostrar texto lido ajuda a enxergar o que o sistema conseguiu capturar.
Depois dá para adaptar também para fornecedor, CNPJ, produto e chave de acesso.
